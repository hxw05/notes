# 我与计算机

这本质上是一篇流水账，但出乎意料地串起了我从小到大对计算机的记忆。这篇文本中有相当多的对细节的记录，可能读起来有些意义不明。

## 2011-2012，一切的开始

### 哈利波特与 Windows XP

2011 年，一年级的我和家人在北京看哈利波特，看的过程中我睡着了。

我接触计算机的起点，是一台放在楼梯下的普通电脑（类似于哈利波特在姨妈家房间的那个位置）。那个位置并不宽敞，也不狭窄，能正好放下一张桌子和一台电脑。对计算机的接触从使用 Windows XP 开始，我从大人那里学会了遇事不决就右键桌面，刷新一下。虽然不知道是在干什么，但是看着桌面上的图标一下子全部闪烁一次，感觉自己好像做出了什么有用的操作似的。

随着我对电脑屏幕上显示内容的“研究”，我开始对这个可以弹出窗口、会有窗口问你确定还是取消以及用生硬的语言写出一些根本看不懂的错误提示的界面整体产生了浓厚的兴趣。吸引我的并不是它的神秘或者对其底层逻辑的探索欲，而是一样最对于**确定性**的向往：点击这个，出现那个，天衣无缝。以及不知为何，我无比希望那些窗口上面能显示出我写上去的字。

我与当时一起玩的小伙伴的一个常驻的游戏是*当老师*。我们会轮流扮演学生和老师，拿着粉笔在家后面的一个房子背面不知何时、为何安放的一块大黑板上写写画画，享受一种别样的乐趣。那个时候我一想起在屏幕上见过的那些页面，就会拿起粉笔在门前的瓷砖上把它们描绘出来，然后想象这个按钮点了以后会发生什么事情，弹出一个什么样的窗口，以及这个窗口上面显示什么文字。就这样画了擦，擦了画，我似乎发明了自己的电脑。后来，我有了自己的黑板。

主力电脑挪了位置，从哈利波特在姨妈家的房间的那个位置移动到了我和爸妈的房间。没错，那个时候上小学，我还没有自己的房间。这个时候，我有更多的机会去体验这个电脑使用起来是什么样的。

### 《愤怒的小鸟》、《粘粘世界》、《机械迷城》、《植物精灵》

一次，我听见我姐在电脑上玩《愤怒的小鸟》，听着游戏声音感觉很带劲，于是后来我也开始自己尝试，体验了许多电脑上能玩的游戏，比如《植物大战僵尸年度版》（我到现在还不知道这个年度版是什么意思）、《粘粘世界》（美术设计非常吸引我的一款游戏，2025 年出了第二部）、《平衡球》（它的画面和配乐实在是过于魔幻，甚至对于那个年纪的我来说有些恐怖，但是我还是坚持玩下去了呃）等等。我还在家隔壁的五金店老板那里听说了《机械迷城》，于是体验，然后就爱上了这款游戏，后来还尝试了同一个工作室出品的《植物精灵》。这些游戏都在我脑海里留下了很深的印象。这些体验也让我成为了 Amanita Design 和 2D Boys（现在是 Tomorrow Corporation）的忠实粉丝，我会在 Steam 上买他们出的游戏。

通过体验这些优秀的单机游戏，我形成了一种电脑上的单机游戏才是真正的“游戏”的印象，也正因此较早地接触了 Steam（2016 年左右）。它们不同于那些在线就能玩的小游戏，也绝对不能和手机上的商业游戏等同，即使今日的这些游戏大都具有跨端属性。我总觉得这些单机游戏有着他们自己的节奏和特质，其中蕴含了一种表达，而不是单单的吸引玩家。

与游戏一同体验的是 360 安全卫士，当时我觉得这是一个特别伟大的软件。结合背景，其实这么说也无妨，当时的 XP 和 Windows 7 确实需要一些保护。我开始探索这个软件为用户提供的一些“方便的快捷功能”，以及其衍生出来的另外一些软件，例如 360 桌面，印象中它可以为你的电脑添加第二个桌面，然后可以下载软件或者游戏到这个桌面上放着，布局有点类似 macOS 的 Launchpad。

### Windows 7，开始鼓捣操作系统

一次偶然的机会，我在网上看到了“另一个版本的 Windows”的截图。这个系统给我的感觉是，操作逻辑没有改变，但界面设计却完全脱胎换骨，不仅更加现代，其透明的设计散发出一种我无法形容的魅力，甚至有些高级感在里面。经过搜索，我了解到这是 Windows 7。我没有思考 XP 和 7 之间究竟是什么样的联系（毕竟这版本号，一个是字母一个是数字，而且不知道是从哪里到了 7...）。总之，我也想要用上这样一个比 Windows XP 更美观更强大，同时操作逻辑还丝毫未变的系统。于是我开始疯狂的搜索，不论教程里写了什么，反正也看不懂，目标是找到一个能够打开后一键送我去新系统的“软件”（后来知道那是安装程序）。

阴差阳错之间，居然真的成功了。看着新的系统，我无法形容我心底里的成就感，迫不及待地体验新界面和新功能。想象着为自己和父母建立了两个系统账户，还各自设置了不一样的开机声音。后来甚至还去和父母炫耀了我把家里的电脑换成了新的系统。我忘记他们有着怎样的反应，但至少是很惊讶。后来在他们的使用过程中好像遇到了一些问题和不习惯，于是将电脑交给隔壁电脑店老板查看，老板说这个系统里面被安插了很多插件，不能要了。就这样，电脑被老板用 Ghost 还原到了他手上的某个祖传的 XP 系统镜像里（也是在这个过程中我看到了系统安装的另一种方法），我一手安装上的 Windows 7 就此落幕。虽然结果相当于是失败了，但是我仍然为体验过一次这个系统感到开心。

## 2014-2015，Command Prompt 与批处理

搬家以后，电脑放在了我的房间里，于是我自然有了更多与电脑接触的时间。从这个时候开始，我与电脑开始建立更加牢固的关系。最开始依然是旧的电脑，日常在上面所做的事情并没有什么变化。后来，一位当时的表姐夫来到我家为我慷慨地（没错，这是我当时脑子里蹦出来的词语！）装上了一台新主机，到现在还清晰记得 CPU 的型号是 i3-6100，这应该是 2015 年的最新 CPU。除了 CPU 我还留意到其它零件，如很小巧的一块东芝 SSD（120GB，当时不觉得小），以及一块更大的机械硬盘，我开始对这些电脑零件产生好奇。用上新电脑的第一个感觉，就是快，太快了，虽然 CPU 仅为 i3。当时的系统已经变成了 Windows 10，跳过了 Windows 8（后来装了虚拟机体验了一下，感觉确实该跳过...）。

在不断研究这个操作系统提供了哪些内置软件以及功能的过程中，我发现了命令提示符 CMD。看名字是一头雾水，进入以后发现大有门道，这就是传说中的 DOS？慢慢地，我开始第一次接触到代码，准确来说这不算是一门编程语言，而是一个古老且简陋的脚本语言：批处理（Batch）。在批处理中，我第一次感受到自己控制流程的感觉，四处打标签，`goto` 满天飞，但最后还真能做出来自己想要的效果，一直以来的念想似乎就这样实现了！而且，这个过程中我感受到了一种前所未有的自由，我可以自由地编排流程和编写提示，并且针对一个假想的“用户”来提供功能和展示设计。那么，我该做个什么工具呢？我思来想去，其实我更想做的是那些“弹出来的页面”，而不是将自己局限于黑框框里面。但是囿于当时的知识水平，我别无他法。后来我把自己的目标定为，做一个更好用的 CMD！这个有点莫名其妙，用脚本语言做 CMD？其实就是预先写好一些语句，组合好，然后让用户输入数字来选择执行。或许这样做也是有优势的，因为有些功能在 GUI 上可能需要点很多下，在这里输入一个数字就可以。

就这样，我一直在一个连浮点数计算都没有（所以你无法用 Batch 做一个命令行内计算器）的脚本语言里面挣扎，试图做出一些真正有用的东西。在这个过程中，我了解到了 `wmic` 指令的存在以及怎样用它去查出电脑上的一些基本信息，例如系统版本、运行内存、CPU 型号等；了解到了 Windows 系统中 C 盘下面目录的基本作用，因为经常造访 `C:/Windows/System32` 和 `C:/Program Files`；了解到了一个叫做 rundll 的东西能干很多事情；了解到了“控制面板”里的功能几乎都有快捷入口（cpl 文件），这一点打破了我的认知，让我认识到软件的内部是可以拆分的；以及注册表的存在。搜索查询的过程中，我了解到了与 Batch 同为脚本语言的 VBS，其语法显然更高级，但无兴趣深入，因为对当时的我来说有些困难。再加上我也不是天天都在电脑上。于是我对计算机的接触就在此停滞不前了。

## 2016，WordPress、Typecho

2016 年前后，在玩 Minecraft 的过程中认识到了许多网友，于是出于跟风的心理模仿他们自己也弄了一个 WordPress 博客网站，虽然不知道要在上面写什么。我的第一个域名也是在那一年注册的。当时只会使用二手乃至三手服务器经销商提供的廉价虚拟服务器，安装也是跟着教程一步步走，并没有什么深入的了解。在反复鼓捣博客网站的过程中，我了解到了 PHP 这门语言的存在。后来接触到的 Typecho 这样一个博客程序让我感受到了轻量和简洁之美。

## 2017，深圳

2017 年我小学毕业，表姐邀请我去深圳，她工作并定居的城市。那是我第一次一个人坐交通工具，跨越一千公里。我不知道我父母当时怎么想。那天我爸早上喊我起来以后，与我一起坐上开往武昌的绿皮，然后坐地铁辗转到武汉站，把我送到检票口。他因为不放心还嘱托了一位路人，让他看好我，然后我一下楼梯就和这位路人散开了。如果换做我，在回去的路上，会不会感到空虚？嗯...先打住这里。总之，虽然当时还没上初一，虽然我只去了不到半个月，但是可以说这是改变我的思想进而改变我的人生的半个月。她带我领略了深圳、带我去腾讯大厦参观了她工作的地点，这一切都为我留下了极为深刻的印象。最为重要的是，看到这一切让我意识到外面的世界是有多么广阔与精彩。这对于只在县城上生活过的我来说是非常震撼的一次旅行。似乎就是从那以后，我逐渐变得善于表达，因为我开始认为“外面的世界”是真的可能有志同道合的人在等着我。我依稀记得回去以后上初一，一节外教课上，来自喀麦隆（初三前后知道的）的外教问我们暑假去了哪里。我站起来近乎自豪地提到深圳，那个时候我的心里清楚，这个教室里没有另外的人知道我当时心里怀揣着怎样的激情以及为什么怀着这样的激情。

由于深圳实在是太吸引我，后面的 2019 年、2023 年、2024 年、2025 年我都去了一次，其实相当于是除去高中三年和 2018 年的每一年。

## 2018，前端

应该是 2018 年的某一天，表姐给我发来了消息，印象中似乎是一个编程教育平台，名为程序媛（写到这里的时候我去搜了一下，这个平台还在）。没错，这个平台是一个为鼓励女性学习技术而设计的编程学习网站。我拿到了表姐给的内测链接，可以在上面免费学习一些编程语言和相关知识。印象中上面的编程语言选项还挺多的，我选择了 HTML/CSS/JS 这样一条路线，想要补一补自己在博客程序的基础——网页方面的知识，在认真学了几天的 HTML 标签以后，我发现自己竟然真的能不费很大力气就写出页面出来！过去的念想终于实现了一大半了。自此我真正意义上接触了前端。2020 年，我在一次实践中了解到了有 Vue 这样的前端框架的存在，体验了以后感觉很方便，从此一发不可收拾，再也没有写过纯的 HTML/JS/CSS，后来发现大家也都是这样的，HTML/JS/CSS 在前端更像是基础以及编译产物的存在。

单写 HTML/CSS/JS，我很快触碰到了静态页面的瓶颈：只适合展示，而不承载任何复杂逻辑或存储，于是开始尝试用 PHP 来做一些对当时的我来说更加“高级”的网页，仿照网上的教程用 mysqli 连接数据库，虽然不知道 MySQL 是啥。当时的我甚至不知道服务端和客户端的区别。在高中的时候第一次接触到了后端的概念，这个概念与前后端分离同时期进入我的脑袋，就这样奠定了之后我开发的基本模式，我开始研究一些常见的界面以及功能（例如登录）是如何实现的。可惜高中并没有多少时间，系统化学习 Web 前后端工程的路没有走下去。

## 2020，疫情

这是一段插入的内容。之前装的电脑到这个时候已经略显卡顿，所以我琢磨着想要自己装一个。当时在贴吧上看到的装机内容也不少，觉得无非就是将一些别人做好的零件拼在一起，没有什么很困难的。事实确实如此，虽然可能会有一些小问题，但最后都是可以跑起来的，最大的困难出在经济上。我不希望硬件被卡在中间，比如对于 CPU 来说，要买就买一个好的。但是当时的我甚至初中都没有毕业，张口就想要一块 Ryzen 7 3700X 确实很奇怪，但这是我经过多次调研以后确定的一块 CPU...另外，我还想要一个显示器，原本的显示器的分辨率只有 1440*900 的分辨率。

根据我后来对我家那个时候的状况的了解，这不算是一种狮子大开口，但是确实对于那个年纪的小朋友来说，很难让人相信他确实需要这么一块 CPU——除非我在那个时候就能把这篇流水账的前面部分一五一十的写出来，甚至还做一些对未来的预测，可能还有一丢丢的说服力。

在经过了一些我不愿意回想起（但忘不掉）的 battle 之后，我得到了我想要的东西，但是它们二者无一存活很久，这在我心中是一个未解之谜。那块 CPU 在用了不到几个月后的一次开机后马上灭掉，再往后就起不来了。至今我仍然相信它是能用的，就是不知道该拿它怎么办。显示器则是在那一年的五月份一次回家，打开后白屏，然后左下角冒起了烟雾，过了一会儿它自己给自己烧了一个洞。

这很邪门，但是我确实不能再要求更多了。

家里电脑在这个时候停摆了一段时间，我的心情很低落。应该是过了几个月，我爸还是带我去县城里的电脑城买了一台 i5-12900F 的板 U 套装（店里的人推荐的，我问有没有 i7，说没有）。不得不说我至今仍然记得他们真的很会忽悠，我相信价格也不是什么好价格。到这个时候我心里却有一种说不出的感动。我不知道我父母当时对于我在接触电脑这方面究竟是怎样的观点，或许是跟我的学习成绩挂钩？学习好就不管，学习不好就管？但似乎在无形之中他们在支持着我，虽然可能不是我理解的那样。

之后便是没有什么动静的高中三年，略过。

## 2023，高考

高考中我取得的成绩是 635 分，新高考一卷，位次记得应该是 7000+，如果不考虑专业的话，这个分数可以上末流的 985。

但我毅然决然地提出要选择计算机专业，正因为这是我一直以来的挚爱与兴趣。想象一下，你恰好有一个这样的兴趣可以作为大学的一门专业来学习，岂不是很令人激动？而那一年，正是张雪峰的计算机黄金就业论甚嚣尘上的时候，于是有相当一部分人无论出于什么原因，与我一样，报考了计算机。这样，填报志愿的时候，往年一些大学的计算机专业的分数线就不再具有参考价值，往往需要将往年分数线提高了十几乃至几十来算作等价。当时的我其实没有什么主见，也没有坚持要去一个什么样的地方（虽然当时的我心中已经明确了什么地方更加适合我的认知以及我的性格）但排除了东北（所以东北大学 985 其实可以进但没有写在志愿上，与我同分的同学去了），最后和我爸一起确定了志愿，按顺序分别填写的是北京邮电大学、西安电子科技大学和武汉理工大学（后面的省略）。

北京邮电往年的分数线已经比较高，所以只是赌一赌的心态，而西电作为一所相当不错的 211 成为了我的首选，然而它的计算机专业当年录取分数线是 636，因一分错过，最后我来到了招生数量庞大的武汉理工大学，就好像被网住的一条鱼。对于这个差一分错过的故事，我看过很多，但没有想到会发生在我自己的身上，并且这一分，我很难不认为它是因为媒体的渲染导致的。我觉得我被舆论害了，但终究还是得怪自己数学没有少错点。武理的计算机学科评级并不咋地，实际上的体验也确实很符合评级，虽然当时刚进的时候还不愿意承认。得知被武理录取的那一天，我应该在四川和父母住在九寨沟附近的酒店里。那天晚上我在手机上不断地搜索着网上对武理的风评，越看越不对劲，索性不看了，至少不坏心情，“至少还是一个 211”。

## 2023，大学

来到大学以后，真切体会到什么叫做“理想丰满、现实骨感”。大一这一年学校的管理较严，并且也没什么专业相关的课程，无非高等数学与政治历史。但庆幸的是我对计算机的热情并未因为环境磨灭，鲜有的两门与计算机相关的课——高级语言程序设计（实际是 C）和程序设计综合实验（用 C 做一个命令行程序，文件读写之类的）我都拿了高分（95 和 99）。部分室友似乎都是跟着张雪峰的话进来的似的，在寝室相处处处受阻的感受让我慢慢感觉看不到生的希望了（用新话来说，就是活人微死）。大一的下学期，因为某些原因我的日子过得非常的灰暗，以至于现在我也不想回忆。也正是那个学期的期末考试，直接奠定了我整个四年绩点的基调，哪怕后面的专业课再怎么赶，绩点也是纹丝不动。也正因此相当于在大一下学期就直接失去了大二家里人才跟我提起的保研的资格。

我没有觉得很可惜，因为这是我的选择，但家里人似乎花了一段时间才接受。

大二换了室友，专业课也变多了，日常感觉明显良好了许多，后面的事情就无需多提了，并不算是这个流水账的一部分，或许我会在我的本科毕业总结里写一写。总的来说，大学并没有像我高中时候天真想象的那样有着无比自由与智慧的氛围，也没有等来“某个富有创造性的同学”能够与我探讨软件设计与 UI。我在这里只看到了大家对绩点、竞赛的关注、毕业或实习之前的临时速成抱佛脚以及老师上课讲的一页又一页的祖传、目的不明的 PPT。我甚至不敢在同学面前提到这些我所认为的这所大学氛围的不足之处。

## 2026，当下

回顾这一路走过来，我觉得我是互联网陪着长大的。在这个过程中，我没有真正的朋友，也没有真正的老师。我感受到了网络的多元性以及广阔性，并且从中了解到了许多我那些年龄根本接触不到的知识。我在这个过程中逐渐建立了语言表达的能力和逻辑思维能力，并且渐渐地形成了自己的性格（当然，后来因为这种长期独处自然带来的孤僻而带来的种种不适应也让我适当地改变、优化了自己的性格）。

我在这里所写的流水账，相当一部分是片面的，因为在这个过程中因为一些思维的限制、发生的事情和其他因素，其实可以说充满了混乱、不堪与跌跌撞撞。但好的一点是，我对计算机的热爱依然没有被磨灭，只是再也回不去初中的时候那种，只要放假，每天从早上醒来开始，可以在电脑上探索、打字直到晚上睡觉的热情，毫不夸张。

我不后悔我在大学选择计算机专业，也不后悔将初高中的课外时间（高中时间极少且高三除外）都花在这个上面。

虽然故事的最后，我从优绩主义的视角看着自己与他人的差距被渐渐抹平、拉开，但我也开始接受自己的独特性。
