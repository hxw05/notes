# `tail -n`, `jq -r` - 提取返回JSON中的特定字段


```sh
make request ... | tail -n +2 | jq '.data.markdownContent' -r | glow
```

- `tail -n +2` 的作用是“去掉输出中的第一行内容”
  - `-n`后接的参数为*number*，它可以是一个正数，也可以是一个负数
  - tail指令的具体含义是，从一个“位置”开始，显示其之后的内容（包含该位置本身），这个位置最常见由`-n`参数（展开形式为 `--lines`）指定
  - number参数的默认符号（不带符号时）是负号，表示倒数第几个单位；`-n 1`就表示倒数第一个单位，且单位为行，于是`tail -n1`就是“显示最后一行”，`tail -n2`就是显示最后的2行，以此类推。
  - number参数通过指定`+`号，也可以接受正数，表示正数第几个单位；`-n +1`表示正数第一个单位，且单位为行，于是`tail -n +1`就是显示从第一行开始的内容且包含第一行，相当于显示所有内容；`tail -n +2`则是除去第一行的全部内容
- jq的`-r`参数用于返回一个raw结果，避免显示颜色符号或引号
