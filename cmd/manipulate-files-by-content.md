# `rg -l`, `xargs` - 根据内容操作文件

Note: 该指令仅适用macOS。

```sh
rg '目标内容' -l | tr \\n \\0 | xargs -0 -I {} rm '{}'
```

- `rg -l`用于只返回文件名称
- `tr \\n \\0`将换行符\n替换为NUL。需要注意NUL是一个确切存在的字符，只是不会显示出来
- `xargs`
  - `-0`参数用来将其分隔符从“换行或空格”替换为NUL，在这里等价于处理之前的换行
  - `-I {}`用来指定一个*replstr*（在这里是`{}`），用来在其形成的指令中自由指定参数位置

`rg -l`之后还可以加grep/grep -v/...用于对路径名过滤。
