# 17 日期和时间

## 创建Date对象

- 直接创建，等于当前时间 new Date()
- 使用毫秒时间戳创建：new Date(number)，时间戳可以是负数，表示1970-01-01 00:00:00之前的时间。
- 使用字符串创建，算法与Date.parse相同
- 指定年月日时分秒毫秒
    - 年可以是四位数或者两位数，如果是两位数则表示19xx，不推荐两位数
    - 月从0开始
    - date默认是1

## 常用方法

以下方法均针对客户端时区

- getDay，获取是星期几。0表示星期日
- getFullYear，获取四位数的年份。不要使用getYear

如果需要在UTC时区中获取，就在get后加上UTC，如getUTCFullYear()。

- getTime，获取毫秒时间戳。
    - 与Number转型等价：Number(new Date())或+new Date()
    - 日期可以相加减，相当于时间戳相加减。
    - Date.now() 可以直接获取当前的时间戳而不需要创建Date对象，推荐使用。
- getTimezoneOffset，获取UTC与本地的时差，**以分钟为单位**。
    - 在中国，new Date().getTimezoneOffset() === -480

## 自动校准

不合理的日期会自动进位。

```jsx
let date = new Date(2013, 0, 32);
// date是2013-02-01
let date = new Date(2016, 0, 2);
date.setDate(0);
// date是2015-12-31
```

闰年会自动处理

```jsx
let date = new Date(2016, 1, 28); // 闰年，二月有29天
date.setDate(date.getDate() + 2);
// date是2016-03-01
```

## Date.parse

Date.parse解析RFC3339格式的字符串

- YYYY-MM-DDTHH:mm:ss.sssZ

不一定要完整填写，最多可以省略到YYYY。Z的地方填写+-hh:mm表示相对UTC的时区偏移，填Z就是UTC。
