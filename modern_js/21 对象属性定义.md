# 21 对象属性定义

## Object.defineProperty

在对象上定义一个属性，并且对其进行配置。

- writable：属性是否可写
- enumerable：属性是否可列。如果不可列，将不会出现在for in循环中和Object.keys的返回值
- configurable：属性是否可以删除、可配置

```jsx
const a = {};
Object.defineProperty(a, 'prop1', {
	value: '123',
	configurable: true,
	enumerable: false,
	writable: false,
});

a.prop1 = '456';
console.assert(a.prop1 === '123');
console.assert(Object.keys(a).length === 0);

for (let k in a) {
	console.log(k);
}
```

不同于对象的赋值和字面定义形式，使用defineProperty定义时，三个属性默认值均为false，而赋值时均为true。

### Object.defineProperties

defineProperty的批量形式

```jsx
Object.defineProperties(a, {
	prop1: {
		value: 'v1',
		// ...
	},
	prop2: {
		value: 'v2',
	}
});

console.log(a);
```

### writable

如果一个字段writable为false，那么对该字段的赋值操作是no-op

### configurable

configurable决定是否能对这个字段进行配置，即调用defineProperty更改其属性和值。使用defineProperty更改值不受writable影响。

当configurable为false时，仍然可以使用defineProperty，但只能将writable从true改为false。

### enumerable

字面含义。

## Object.getOwnPropertyDescriptor(s)

获取某个属性或者全部属性的的descriptor（描述器）。描述器是一个专门的对象。

## 访问器（accessor）

使用defineProperty还可以定义访问器，即getter和setter，这两个函数的固定名称为get和set。注意getter和setter与value和writable不能混用。

```jsx
Object.defineProperty(obj, 'name', {
	set(v) {
		const parts = v.split(' ');
		this.lastName = parts[0];
		this.firstName = parts[1];
	},
	get(v) {
		return this.lastName+this.firstName;
	},
	value: 'abc',
	writable: false,
}

```
