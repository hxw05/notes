# 写在前面：重学JavaScript

## 主题

这一组笔记主要记录了我跟着 Modern JavaScript Tutorial https://javascript.info 在线文档（大部分看的是英文版，因为我看的时候感觉中文的翻译质量不佳）所复习的JS基础，内容很杂（但也按照主题标号分类了），就跟JS这门语言一样。

之所以需要重新复习一次JavaScript，一是为了应付面试，二是为了让自己对JS这门语言有更完整的了解，尽可能去摆脱一种写出来的代码似懂非懂的感觉，也让读AI写的代码的时候更容易也更有自信。

## 关于JS

ES6以及之后的JS比之前更像一门严肃的高级语言（虽然有着独特于其它所有语言的特殊设计），而不是某种纸糊的脚本语言。ES6之前的ES5给我的感觉就很混乱（所以当时其实根本就没有学下去），单就几个社区造出来的模块系统就让人不知道从哪里入手。

教程中提及的一些ES6+特性，是我看了之后才知道。例如`Symbol`到底有什么用。但说实话，在我没有看到这里之前，我也从来没有想过会用到它（除了Vue的文档中可能用它做过标识符）；看过之后，虽然觉得用`Symbol.iterator`、`Symbol.isConcatSpreadable`等等去“实现”一些对象上的特性很酷、很先进也很方便，但就是觉得在实际中，尤其是工作中，很难用到（除非leader知道且愿意去用）。不过我觉得如果要自己写一个Library的话，大概率是会用上`Symbol.toStringTag`的。

## 重点

如你所见，JS的内容很多也很杂，就像高中的时候背文科一样，没有办法。我觉得面对这种性质的知识，提取重点是很重要的，尤其是提取那些自己过去非常希望搞懂、在实际中也经常用到的主题。下面是我为自己划的重点以及原因，供未来反复翻阅。
