# 06 函数

JS具有一定函数式语言特性，在这里函数被看作是值。有两种创建函数的方式：

- 使用function关键字声明的函数，在其**所在的语句块内部**被定义之前也可以使用。
- 使用赋值语句定义的函数，在定义之前不能使用。

```jsx
let age = prompt("What is your age?", 18);

if (age < 18) {
	// 仅当前块
  function welcome() {
    alert("Hello!");
  }

} else {
	// 仅当前块
  function welcome() {
    alert("Greetings!");
  }
}

welcome(); // Error: welcome is not defined
```
