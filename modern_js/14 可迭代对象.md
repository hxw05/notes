# 14 可迭代对象

## 迭代器

可迭代对象是可以使用for … of迭代的一种对象。for … of循环开始时，会先调用对象上的Symbol.iterator创建一个迭代器。Symbol.iterator处存储的是一个返回**迭代器（iterator）**的函数。

一个迭代器是一个对象，它上面至少有一个next方法总是返回 {done: boolean, value: any}，当done=true时表示循环结束，value表示下一个循环的值。

可以模仿for … of的流程，手动调用迭代器完成迭代。

```jsx
let str = "Hello";

let iterator = str[Symbol.iterator]();

while (true) {
  let result = iterator.next();
  if (result.done) break;
  console.log(result.value); // 一个接一个地输出字符
}
```

## 可迭代对象（Iterable (Object)）和类数组（Array-like）

- 一个可迭代的对象，或者称为一个iterable，是一个实现了Symbol.iterator的对象。
- 一个类数组是一个带有索引和length属性的对象

一个可迭代对象可能不是类数组，一个类数组也可能不可迭代。

字符串是典型的可迭代对象和类数组。

### Array.from

:::tip
由于字符串可迭代也是类数组，所以可以用Array.from从字符串创建一个字符数组。
:::

Array.from用于创建一个iterable或者array-like对应的array。

```jsx
let arrayLike = {
  0: "Hello",
  1: "World",
  length: 2
};

let arr = Array.from(arrayLike);
```

Array.from除了第一个参数外，还有

- mapFn，在元素被添加到被创建的数组之前，先对该元素进行变换。
- thisArg，用于设置mapFn的this
