# 39 动态导入

```jsx
export function hi() {
  alert(`Hello`);
}

export function bye() {
  alert(`Bye`);
}
```

```jsx
let {hi, bye} = await import('./say.js');

hi();
bye();
```

import()用于动态导入，它是一个特殊的函数形式语句（但不是一个函数），返回一个包含相关模块所有导出项目的Promise，可以用await来获取其内容。默认导出对应其返回值的default字段。

动态导入既可以在模块中使用，也可以在一般的脚本中使用，也就是说不需要加type=module。
