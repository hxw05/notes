# 02 数据类型，类型转换

目前的JS中有7种基本数据类型（primitive）和1种引用数据类型。JS是动态类型的语言，变量的数据类型不是固定的。

## Number

JS中标准的Number是一个double64，即64位双精度浮点型。

- Number.MAX_SAFE_INTEGER == 2^53-1
- Number.MIN_SAFE_INTEGER == -2^53-1
- 1/0 === Infinity
- `isNaN("a"/2) === true`

### NaN

一般NaN会导致传播，但是有一个例外：NaN ** 0 === 1

在JS中进行的数学运算永远不会导致产生结果以外的事情，最坏情况也会产生NaN。

## BigInt

BigInt可以表示**任意长度的**整数。在**整数**的末尾加上n表示这是一个BigInt。

## null

null是一个单独的类型，**但是 typeof null的结果是object**，这是一个固定的行为，也是一种固定的错误。

## 基本类型转换

基本类型的转换一般使用对应的构造函数进行，如用String转换为string类型，Number转换为number类型。

到number的类型转换会自动发生在算术表达式中，例如

- `"6" / "3" === 2`

注意：加法是特例，见 [03 基础运算符](03%20%E5%9F%BA%E7%A1%80%E8%BF%90%E7%AE%97%E7%AC%A6%202f8b76c581cf80358a76d245a402b7f3.md) 

number转换中，

- undefined被转换为NaN，**而null被转换为0**（而不是NaN）
- 对于字符串，在转换之前可以看作有一层trim，之后
    - **如果是空字符串**，被转换为0
    - 如果不是纯数字字符串（numeric string），被转换为NaN
    - 被转换为对应的数字

boolean转换中，

- falsy的值被转换为false，truthy的值被转换为true
- falsy的值是指：
    - 数字0
    - **空字符串**
    - null
    - undefined
    - NaN
- 其余的值均为truthy

注意boolean转换过程没有trim，因此非空的空格字符串（non-empty blank string）会被转换为true。
