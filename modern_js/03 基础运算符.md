## 加法/连接运算符 +

- 任意一个运算元是字符串，其余运算元都会被转换为字符串。

```jsx
console.assert("1" + 2 === 2 + "1")
console.assert("1" + 2 === "12")
```

- 运算顺序不受运算元的影响。以下语句中，2+2会先被执行，然后再转换成字符串。

```jsx
console.assert(2+2+'1'==='41') // 不是 '221'
console.assert('1'+2+2==='122') // 不是 '14'
```

## 正号（一元运算符）+

在一个数字上使用正号没有明显的作用，在一个非数字上使用正号会将其转型为数字，因此相当于是Number转换的简略形式。

```jsx
let form = {
	fieldA: '2',
	fieldB: '3'
}

console.assert(+form.fieldA+ +form.fieldB===5)
// 注意，两个+不能连写避免与自增混淆。
```

使用以上方式可以不用写Number就得到两个数字字符串对应的数字运算值。一元运算符的优先级高于二元运算符。
