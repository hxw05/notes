# 36 生成器函数

生成器函数使用function* 定义

```jsx
function* generatorFunc() {
	yield 1;
	yield 2;
	return 3;
}

const generator = generatorFunc();
generator.next(); // {done: false, value: 1}
generator.next(); // {done: false, value: 2}
generator.next(); // {done: true, value: 3}
generator.next(); // {done: true, value: undefined}
// ...
```

## 生成器函数的返回值是一种可迭代对象

每一个yield输出的值相当于是Iterable的每一项，但不包含return

```jsx
function* generateSequence() {
  yield 1;
  yield 2;
  return 3;
}

let generator = generateSequence();

for (let value of generator) {
  alert(value); // 1, 2
}
```

适用于多有可迭代对象可进行的操作，包括 for…of、rest等

## Symbol.iterator

在之前，为了制作一个range迭代器，需要我们手写next函数。

```jsx
let range = {
  from: 1,
  to: 5,

  [Symbol.iterator]() {
    return {
      current: this.from,
      last: this.to,
      
      next() {
        if (this.current <= this.last) {
          return { done: false, value: this.current++ };
        } else {
          return { done: true };
        }
      }
    };
  }
};
```

使用generator语法改写以后：

```jsx
let range = {
  from: 1,
  to: 5,

  *[Symbol.iterator]() {
    for(let value = this.from; value <= this.to; value++) {
      yield value;
    }
  }
};
```

## 组合生成器

yield* 语法可以在一个生成器函数中，“嵌入”另一个生成器的输出结果。

```jsx
function* generateSequence(start, end) {
  for (let i = start; i <= end; i++) yield i;
}
```

```jsx
function* generatePasswordCodes() {
  // 0..9
  yield* generateSequence(48, 57); // 相当于在这里有那么几行yield，下同
  // A..Z
  yield* generateSequence(65, 90);
  // a..z
  yield* generateSequence(97, 122);
}
```

## 用yield获取外部值

```jsx
function* gen() {
  let ask1 = yield "2 + 2 = ?";

  alert(ask1); // 4

  let ask2 = yield "3 * 3 = ?"

  alert(ask2); // 9
}
```

yield本质上是一个表达式，它的值是外部调用next时传入的第一个参数。

生成器实例的第一个next调用一定是不带参数的（如果带了则忽略），因为第一个yield还没有来得及执行。

调用了一次next后，第一个yield执行并会返回出yield后紧跟的表达式的值。这时如果调用next并传入值（如果不传入则代表undefined），该值会作为函数内部该yield处整体表达式的值。

上面gen函数的调用过程如下：

```jsx
const g = gen();
g.next(); // first call, "2 + 2 = ?"
g.next(4); // yield "2 + 2 = ?" 取值为4，并返回value="3 * 3 = ?",done=false
g.next(9); // yield "3 * 3 = ?" 取值为9，并返回value=undefined,done=true
```

## generator.throw

在生成器内当前yield处抛出一个错误，而非取值。

```jsx
function* gen() {
  try {
    let result = yield "2 + 2 = ?"; // (1)

    alert("The execution does not reach here, because the exception is thrown above");
  } catch(e) {
    alert(e); // shows the error
  }
}

let generator = gen();

let question = generator.next().value; // 2 + 2 = ?

generator.throw(new Error("The answer is not found in my database"));
// 导致 yield "2 + 2 = ?" 抛出该错误
```

如果在生成器中没有对相关错误的处理，那么该错误将会穿出生成器到达当前行。

## generator.return

提前停止生成器的执行，并强制使其返回一个给定的值。

```jsx
function* gen() {
  yield 1;
  yield 2;
  yield 3;
}

const g = gen();

g.next();        // { value: 1, done: false }
g.return('foo'); // { value: "foo", done: true }
g.next();        // { value: undefined, done: true }
```
