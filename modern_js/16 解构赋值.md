# 16 解构赋值

## 基本语法

```jsx
let [firstName, , title] = ["Julius", "Caesar", "Consul", "of the Roman Republic"];
```

可以添加逗号来忽略不需要的那些值。

- 解构赋值语法的右侧可以是任意可迭代对象，例如字符串，不一定要是数组。相当于在右侧值上调用for … of并分别赋值。
- 左侧可以是任意可以赋值的内容，不仅限于变量，也可以是对象的属性。

### 先声明后解构

需要这样写来避免{…}被看作是代码块。

```jsx
let title, width, height;

// 括号
({title, width, height} = {title: "Menu", width: 200, height: 100});
```
