# missing_2020

这里记录的是我从2025年3月开始看的MIT课程《The Missing Semester》的笔记。这门课以YouTube视频的形式呈现。由于是边看边记的，可能会比较杂乱。我在CSDIY上听说了这门课。

这门课可以说*有点年份*了（2020年的），其中包含了许多平时经常用到却没有时间去专门搞懂的知识。这就不得不提，我喜欢macOS的一大原因就是它的Unix-like属性，让我不需要研究Windows上的两个命令行怪胎（指cmd和powershell，我承认后者很强大），而是去拥抱开源社区甚至一些CS课程（教师）的标准——Linux下的shell。当然，我觉得我这边本科期间的专业课老师大概可能只会用`ipconfig`之类的吧...

总之，这门课的重点便是shell。五六年时间过去了，现在的coding agent中依然随处可见这些Linux基础指令的身影，它们是agent们的工具——`rg`、`grep`、...，可见这门课的前瞻性。正如其名字，这门课本意是给CS学生“查漏补缺”用的。
